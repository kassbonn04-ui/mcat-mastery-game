# Art credits — Arcanum / MCAT Mastery

## Hard rule (public GitHub Pages)

**Do not** download, commit, or redistribute:

- Warner Bros / Pottermore / Wizarding World official art
- Movie stills, screenshots, promotional posters, or trademarked logos
- Celebrity likenesses of film actors

Those remain copyrighted. This study game uses **Arcanum** framing with Hogwarts-*inspired* place parallels and **licensed photographic / classic illustration** atmosphere only.

If Kass later supplies personally owned assets, they can replace files under `assets/` without changing licenses of third-party packs.

### Trial strip pack (`assets/trial-atmos/`)

Used only on figure-less question headers (“Sage watches” strip). Homepage / companion
screens keep `assets/hero-castle.png` + fireflies. Plates include project art
(`spell-chamber`, `world-map`) plus photographic campus moods (canal boat, shelf alley,
library, steam train, mist keep, star ferry). Light CSS motion is decorative only.
